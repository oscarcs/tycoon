Rendering progress

- Built a Vite/React browser game with a canvas isometric renderer.
- Loads source_tileset_manifest.csv and source_tileset PNGs through public symlinks so the existing art remains authoritative.
- Uses the outlined empty tile as the default base tile when available.
- Renders water, coastline tint, farms, forests, residential/commercial/industrial buildings, purchased land hatching, roads, rail, crossings, stations, depots, and smoothly interpolated trains.
- Renders elevated rail, bridges, monorail guideways, ports, and airport runways with procedural fallback graphics.
- Renderer keeps tile height in the model for future elevation work.
- Palette swapping is represented in tile state with palettePhase and the renderer is centralized so a later indexed-palette shader/pass can be added in one place.
